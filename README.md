## ğŸ¯ Tentang Owi-MD

Owi-MD adalah bot WhatsApp berbasis Baileys yang dirancang dengan arsitektur modern dan sistem plugin yang fleksibel. Bot ini dikembangkan oleh SawitProject untuk memberikan pengalaman penggunaan yang optimal dengan berbagai fitur menarik mulai dari hiburan, manajemen grup, hingga alat produktivitas.

Bot ini dibangun dengan mempertimbangkan kemudahan penggunaan dan kemampuan kustomisasi yang tinggi, sehingga pengguna dapat dengan mudah menambahkan fitur baru sesuai kebutuhan tanpa perlu memodifikasi kode inti. Dengan dukungan komunitas yang aktif, Owi-MD terus mengalami pengembangan dan peningkatan fitur secara berkala.

Keunggulan utama dari Owi-MD terletak pada stabilitas koneksi, respons yang cepat, serta dokumentasi yang lengkap untuk membantu pengguna dalam menggunakan dan mengembangkan bot ini. Selain itu, bot ini juga mendukung berbagai mode operasi yang dapat disesuaikan dengan kebutuhan, baik untuk penggunaan pribadi maupun komersial.

## ğŸš€ Cara Instalasi

### Metode 1: Instalasi di VPS/Linux Server

Ikuti langkah-langkah berikut untuk menginstal Owi-MD di server Linux Anda:

```bash
# 1. Masuk ke server dan update sistem
sudo apt update && sudo apt upgrade -y

# 2. Install Node.js 20.x
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 3. Install Git
sudo apt install git -y

# 4. Clone repository
git clone https://github.com/SawitProject/owi-md.git
cd owi-md/wowo

# 5. Install dependencies
npm install

# 6. Jalankan bot
npm start
```

### Metode 2: Instalasi di PC/Lokal

Jika Anda ingin menjalankan bot di komputer lokal:

```bash
# 1. Install Node.js 20.x dari https://nodejs.org/

# 2. Download dan ekstrak source code
git clone https://github.com/SawitProject/owi-md.git
cd owi-md/wowo

# 3. Install dependencies
npm install

# 4. Jalankan bot
npm start
```

### Metode 3: Menggunakan Panel Hosting

Untuk pengguna panel seperti Pterodactyl atau panel lainnya:

1. Upload seluruh folder `owi-md/wowo` ke panel
2. Pada menu Startup, pastikan Docker Image adalah Node.js 20
3. Pada kolom Startup Command, masukkan: `node index.js`
4. Start server dan bot akan berjalan otomatis

## ğŸ”Œ Sistem Plugin

Owi-MD menggunakan sistem plugin modular yang memungkinkan Anda menambah atau mengubah fitur tanpa memodifikasi kode inti. Setiap plugin adalah file JavaScript independen yang dapat di-load secara dinamis.

### Struktur Plugin

```javascript
// Nama plugin: contoh-plugin.js

export default {
  // Nama command yang ditangani
  Commands: ["contoh", "example"],
  
  // Deskripsi plugin
  description: "Contoh plugin untuk demonstrasi",
  
  // Jika true, hanya premium yang bisa menggunakan
  OnlyPremium: false,
  
  // Jika true, hanya owner yang bisa menggunakan
  OnlyOwner: false,
  
  // Jika true, hanya admin grup yang bisa menggunakan
  OnlyAdmin: false,
  
  // Jika true, hanya bisa digunakan di grup
  OnlyGroup: false,
  
  // Berapa limit yang dikurangi per penggunaan
  limitDeduction: 1,
  
  // Fungsi utama handler
  async handle(sock, messageInfo) {
    const { remoteJid, message, prefix, command } = messageInfo;
    
    // Logika plugin di sini
    await sock.sendMessage(remoteJid, {
      text: "Contoh plugin berhasil dijalankan!"
    }, { quoted: message });
  }
};
```

### Cara Menambah Plugin Baru

1. Buat file baru di folder `wowo/plugins/` dengan nama deskriptif, misalnya `wowo/plugins/EXAMPLE/contoh.js`
2. Ikuti struktur plugin di atas
3. Restart bot untuk memuat plugin baru
4. Plugin akan otomatis terdaftar di menu

## ğŸ”‘ API Key

Owi-MD menggunakan API dari Autoresbot untuk beberapa fitur premium. Untuk mendapatkan API Key:

1. Kunjungi https://autoresbot.com/services/rest-api
2. Daftar akun dan pilih paket yang diinginkan
3. Dapatkan API Key dari dashboard
4. Masukkan API Key pada konfigurasi di `wowo/config.js`

**Catatan:** API Key bersifat opsional. Tanpa API Key, bot tetap dapat berjalan dengan fitur dasar, namun beberapa fitur premium mungkin terbatas.
